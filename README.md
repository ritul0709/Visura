# Visura  
AI Skin and Eye Care Platform  

---

## ⚙️ How It Works  

### 1. **Live Scan**  
- Click **Start Live Scan**  
- Allow camera access  
- The AI model analyzes the webcam feed in real time  

### 2. **Image Analysis**  
- Upload an image  
- Describe the issue (redness, itching, blurry vision, etc.)  
- AI provides basic insights and guidance  

### 3. **Chat Assistant**  
- Ask about symptoms  
- Get help with navigation or booking appointments  

---

## 🧠 AI & ML Fundamentals Used  

- **Supervised Learning:** The model is trained on labeled image data to recognize patterns in skin and eye conditions  
- **Image Classification:** Uses Machine Learning to classify input images into different condition categories  
- **Pattern Recognition:** Identifies visual features like color, texture, and shape for prediction  
- **Model Inference:** Runs predictions in real time directly in the browser  
- **Client-Side AI:** Ensures fast processing without relying on external servers  

---

## 🧪 AI Model  

- Built using **Google Teachable Machine**  
- Uses **image classification (Machine Learning)**  
- Runs entirely in the browser  
- No user data is stored or sent to any server  

---

## 🔐 Privacy  

- No login required  
- No data storage  
- Camera access is fully browser-controlled  
- All processing happens client-side
